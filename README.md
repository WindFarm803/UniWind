UniWind is a unified framework for short-term wind power forecasting that enables accurate predictions across diverse wind farms without site-specific retraining. It supports zero-shot deployment, maintains stable performance on unseen sites, and improves reliability under challenging conditions such as extreme ramp events and low-power periods.
